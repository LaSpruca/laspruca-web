BF in Rosetta

Brainf**k in many langauges

2021-01-04

This is a collaboration project between my self and Jasper (Fallstop), making too many brainf**k interpreters. A full list of currently functional languages is available on the GitHub repo. It started after we had a competition to make the fastest interpreter in rust. Then we made a PHP version, running on an old tablet, available here. We then made a few more. You can add some of your own if you want as well.
